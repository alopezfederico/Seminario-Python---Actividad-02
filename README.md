## Actividad 2 - Entregable

Este repositorio desarrolla la Actividad entregable Nro 2.

### Organización del proyecto
- Proyecto 
    - Carpeta: Notebooks  
    - Carpeta: src  
    - requirements.txt  
    - README.md

Dentro del directorio **Notebooks** se podrán encontrar los archivos con formato .ipynb (Jupyter Notebooks) que desarrollan los ejercicios solicitados en la Actividad 2

Dentro del directorio **Src** se encuentran los archivos .py que contienen las funciones que utilizan los notebooks correspondientes para resolver las consigans del TP.

El archivo **requirements.txt** sirve para poder descargar las dependencias utilizadas en este proyecto.
Para instalar las dependencias ejecutar en consola (preferentemente en entorno virtual):   
    
    pip install -r requirements.txt
